## 🎯 操作指南

### 基础操作
- **WASD / 方向键**: 移动
- **E / 空格**: 交互/使用工具
- **I**: 打开背包
- **ESC**: 暂停菜单

### 工具切换
- **1-6**: 快速选择工具
- **Q**: 切换到上一个工具
- **E**: 切换到下一个工具

### 工具说明
1. **锄头**: 耕地，将草地变成可种植的土壤
2. **水壶**: 给作物浇水
3. **斧头**: 砍伐树木
4. **镐子**: 开采矿石
5. **镰刀**: 收割作物
6. **鱼竿**: 钓鱼

## 🏗️ 项目结构

```
pastoral-tales/
├── assets/           # 游戏资源
├── src/             # 源代码
│   ├── autoload/    # 全局单例
│   ├── core/        # 核心系统
│   ├── entities/    # 游戏实体
│   ├── ui/          # 用户界面
│   └── world/       # 世界地图
├── data/            # 数据配置 (JSON)
└── docs/            # 文档
```

## 🚀 运行项目

1. 安装 Godot 4.2+
2. 打开项目文件夹
3. 运行 `project.godot`
