# 下载的资源

此目录包含从网络下载的开源游戏资源。

## 许可说明

下载的资源可能受不同许可证约束，请查看原始来源:
- LPC 资源: CC-BY-SA 3.0 / GPL 3.0
- Kenney 资源: CC0 (公共领域)

## 使用建议

1. 检查每个资源的许可证
2. 确保符合项目需求
3. 必要时替换为自定义资源

## 手动下载推荐

如果自动下载失败，请手动访问:
- https://opengameart.org/ (搜索 "farming", "crops", "pixel art")
- https://kenney.nl/assets (免费资源)
- https://itch.io/game-assets/free (免费资源)
