# 🎮 设计文档

## 游戏概述

**田园物语 (Pastoral Tales)** 是一款2D像素风农场模拟RPG游戏，灵感来自星露谷物语。玩家继承祖父的旧农场，通过种植作物、饲养动物、与村民建立友谊，逐步将荒废的农场变成繁荣的家园。

## 核心玩法循环

```
早晨起床 → 查看天气/日历 → 农场工作(浇水/种植/收获)
    → 探索/社交/任务 → 夜晚回家 → 睡觉存档 → 新的一天
```

## 游戏特色

### 🌾 农场经营
- **种植系统**: 多种作物，不同季节，生长周期
- **动物饲养**: 鸡、牛等动物，产出蛋奶
- **工具升级**: 锄头、水壶等工具可升级提高效率
- **农场扩建**: 随着时间解锁更多土地

### 👥 社交系统
- **NPC日程**: 每个NPC有自己的日常安排
- **好感度**: 通过对话、送礼提升关系
- **节日活动**: 季节性节日，特殊事件
- **任务委托**: NPC发布的各种任务

### 🎣 探索内容
- **钓鱼系统**: 多种鱼类，不同地点，钓鱼小游戏
- **矿洞探险**: 多层矿洞，挖掘矿石，战斗怪物
- **采集系统**: 野外采集资源、蘑菇、野花

### 🍳 生活系统
- **烹饪系统**: 使用食材制作料理恢复体力
- **制作系统**: 合成工具、装备、装饰品
- **房屋升级**: 扩建房屋，购买家具装饰

## 系统设计

### 时间系统
- **游戏时间**: 现实1秒 = 游戏1分钟
- **一天时长**: 游戏20小时 (6:00 - 2:00)
- **季节周期**: 每季28天，四季循环
- **年份**: 每年4季，共112天

### 体力系统
- **体力消耗**: 使用工具、战斗消耗体力
- **恢复方式**: 吃食物、睡觉、泡温泉
- **体力耗尽**: 无法工作，可能晕倒

### 天气系统
- **晴天**: 正常游戏
- **雨天**: 无需浇水，特定鱼类出现
- **雪天**: 冬季特有，影响作物生长

### 经济系统
- **收入来源**: 出售作物、动物产品、鱼类、矿石
- **支出项目**: 购买种子、工具升级、建筑扩建
- **价格波动**: 部分物品价格随机波动

## 技术规格

### 引擎
- **Godot 4.2+**
- **GDScript**
- **2D像素风格**

### 分辨率
- **游戏窗口**: 1280x720
- **像素艺术**: 16x16 / 32x32

### 平台
- PC (Windows/Mac/Linux)

## 美术风格

### 色彩方案
- **春季**: 明亮、绿色、粉色
- **夏季**: 鲜艳、黄色、蓝色
- **秋季**: 温暖、橙色、棕色
- **冬季**: 冷淡、白色、灰色

### UI设计
- **复古风格**: 像素边框、简洁图标
- **信息清晰**: 体力条、时间、金钱一目了然
- **反馈及时**: 收获、送礼等操作有视觉反馈

## 音效设计

### 背景音乐
- **农场**: 轻松愉快的乡村音乐
- **村庄**: 温馨的社区氛围
- **矿洞**: 神秘紧张的节奏
- **海滩**: 宁静的海浪声

### 音效
- **工具**: 锄头、水壶等使用音效
- **收获**: 作物收获的清脆声音
- **钓鱼**: 抛竿、咬钩、收杆音效
- **UI**: 按钮点击、菜单打开关闭
