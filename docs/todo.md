### 详细任务描述

#### ⬜ P1-T1: 项目初始化
**描述：** 创建 Godot 4.x 项目基础结构

**具体步骤：**
1. 创建 Godot 4.x 新项目，命名为 "PastoralTales"
2. 设置项目配置：
   - 窗口大小：1280x720
   - 拉伸模式：canvas_items
   - 像素艺术导入预设
3. 创建项目目录结构（见 architecture.md）
4. 创建基础场景：
   - Main.tscn (主场景)
   - Farm.tscn (农场地图)
5. 设置版本控制：创建 .gitignore

**验收标准：**
- [ ] 项目能在 Godot 中打开
- [ ] 运行后显示空白窗口无报错
